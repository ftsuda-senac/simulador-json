Gerenciamento e Simulador de JSON

Simulador de API com JSON

Objetivo do projeto

Possibilitar o cadastro qualquer JSON válido para ser acessado através de uma API REST

Este projeto foi desenvolvido pensando em facilitar a integração de aplicações frontend que se comunicam com aplicações backend, principalmente nas funcionalidades de consulta de dados

Tecnologias usadas

  • Spring Boot 3.4.5 + Java 21 + Maven
  • Banco de dados H2
  • Thymeleaf
  • Bootstrap 5.3.5 e Font Awesome 6.7.2 acessados via Webjars

Como usar

  1. Acessar a URL: http://localhost:8080/gerenciamento
  2. Cadastrar um JSON
  3. Acessar a URL do JSON cadastrado
  4. Consultar o JSON através de uma aplicação frontend (exemplo: Javacript com Fetch API ou XHR, Angular, React, Vue.js)

Banco de dados H2

Essa aplicação usa o banco de dados H2 que é carregado junto com a aplicação.

Por padrão, no Windows, o arquivo será gravado no arquivo C:\Usuarios\[nome-do-usuario-windows]\tads-pw-jsonserverbd.mv

Para acessar o console de gerenciamento do banco para rodar comandos SQL, acessar a URL http://localhost:8080/gerenciamento e preencher os dados abaixo:

  • JDBC URL: jdbc:h2:file:~/tads-pw-jsonserverbd
  • User name: sa
  • Password: (Deixar em branco)
